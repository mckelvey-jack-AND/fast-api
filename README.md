# ONS POC

### How to run the FastAPI

```
$ cd backend
$ python3 -m venv env
$ source env/bin/activate
(env)$ pip install -r requirements.txt
(env)$ python main.py
# Depending on your python version you may need to run the following command instead.
(env)$ python3 main.py
```

### How to run the ReactJS

```
$ cd frontend
$ npm install
$ npm run start
```
